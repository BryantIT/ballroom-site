## 1. Account Management

**US-001**
As a new visitor, I want to register with my email and password so that I can create a personal dance profile.

**US-002**
As a registered user, I want to sign in with my email and password so that I can access my saved data across devices.

**US-003**
As a registered user, I want to sign in with Google or Apple so that I can log in quickly without remembering a password.

**US-004**
As a registered user, I want to edit my profile (name, photo, home studio, bio) so that my account reflects who I am.

**US-005**
As a registered user, I want to change my email address or password so that I can keep my credentials up to date.

**US-006**
As a registered user, I want to delete my account and all associated data so that I have full control over my personal information.

**US-007**
As a registered user, I want my session to stay active on my phone so that I don't have to log in every time I open the app.

## 2. Dance Catalog & Levels

**US-010**
As a dancer, I want to browse a catalog of standard and Latin ballroom dance styles so that I can find the dances I am learning.

**US-011**
As a dancer, I want to add a dance style to my profile so that I can start tracking my progress in that style.

**US-012**
As a dancer, I want to remove a dance style from my profile so that I can keep my list relevant to my current studies.

**US-013**
As a dancer, I want to assign a syllabus level to each dance (e.g., Bronze, Silver, Gold, Open) so that my tracker reflects where I currently stand.

**US-014**
As a dancer, I want to update my level for a dance when I advance so that my profile always shows my current standing.

**US-015**
As a dancer, I want to see the full progression path (all levels) for each dance style so that I know what lies ahead.

**US-016**
As a dancer, I want to filter my dashboard by dance style or level so that I can focus on one area at a time.

---

## 3. Pattern Progress Tracking

**US-020**
As a dancer, I want to see a list of syllabus patterns for my current level in each dance so that I know exactly what I should be working on.

**US-021**
As a dancer, I want to mark a pattern as "learning," "working," or "mastered" so that I can see my progress at a glance.

**US-022**
As a dancer, I want to add personal notes to a pattern (e.g., coaching tips, corrections) so that I can recall key feedback during practice.

**US-023**
As a dancer, I want to rate my confidence on a pattern from 1–5 stars so that I have a quick visual summary of my weak spots.

**US-024**
As a dancer, I want to mark a pattern as a "focus" item so that it appears prominently on my practice dashboard.

**US-025**
As a dancer, I want to see a progress bar per dance showing how many patterns I have mastered out of the total for my level so that I feel motivated by visible progress.

**US-026**
As a dancer, I want to view a history of status changes for each pattern so that I can see how long it took me to master it.

---

## 4. Practice Logging

**US-030**
As a dancer, I want to log a practice session with date, duration, and dances worked on so that I can see my effort over time.

**US-031**
As a dancer, I want to attach specific patterns practiced to a session log entry so that I have a detailed record of each session.

**US-032**
As a dancer, I want to add free-form notes to a practice session so that I can capture what felt good or what needs work.

**US-033**
As a dancer, I want to view a weekly and monthly practice summary (total hours, dances, sessions) so that I can see whether I am hitting my practice goals.

**US-034**
As a dancer, I want to log whether a session was solo practice, group class, or private lesson so that I can see the breakdown of my training types.

---

## 5. Progress Reports

**US-040**
As a dancer, I want to generate a progress report for a single dance style so that I can review everything I have learned in that style.

**US-041**
As a dancer, I want to generate a full progress report across all my dances so that I have a comprehensive overview of my dancing.

**US-042**
As a dancer, I want to set a date range for a report so that I can review progress over a specific period.

**US-043**
As a dancer, I want to print or export a report to PDF so that I can share it with my instructor or keep a physical copy.

**US-044**
As a dancer, I want the printed report to be clean and formatted (studio branding optional) so that it looks professional.

---

## 6. Gamification & Achievements

**US-050**
As a dancer, I want to earn achievement badges for milestones (e.g., "First Pattern Mastered," "Bronze Complete," "30-Day Streak") so that I feel rewarded for my progress.

**US-051**
As a dancer, I want to see all available achievements and which ones I have earned so that I am motivated to chase the ones I have not unlocked yet.

**US-052**
As a dancer, I want to receive an in-app notification and celebration animation when I unlock an achievement so that the moment feels special.

**US-053**
As a dancer, I want to earn experience points (XP) for logging practice, mastering patterns, and completing levels so that there is a continuous sense of progression.

**US-054**
As a dancer, I want to see my overall dancer level and XP bar so that I always know how close I am to the next level.

**US-055**
As a dancer, I want to earn streak badges for consecutive days or weeks of logged practice so that I am encouraged to stay consistent.

**US-056**
As a dancer, I want to unlock special titles (e.g., "Bronze Waltz Graduate," "Latin Enthusiast") that appear on my profile so that I can express my journey.

**US-057**
As a dancer, I want a leaderboard among dancers I follow so that friendly competition motivates me to practice more.

---

## 7. Social Sharing

**US-060**
As a dancer, I want to share an achievement card to Instagram, Facebook, or X so that I can celebrate milestones with my dance community.

**US-061**
As a dancer, I want the shareable achievement card to be visually appealing and branded so that it looks great on social media.

**US-062**
As a dancer, I want to share a progress milestone (e.g., "Just completed Silver Foxtrot!") as a pre-formatted post so that sharing takes only one tap.

**US-063**
As a dancer, I want to control which achievements are public vs. private so that I only share what I am comfortable with.

**US-064**
As a dancer, I want to copy a shareable link to my public dance profile so that I can paste it in messages, bios, or studio directories.

---

## 8. Goals & Milestones

**US-070**
As a dancer, I want to set a goal for a dance (e.g., "Master all Silver Cha Cha patterns by October") so that I have a target to work toward.

**US-071**
As a dancer, I want to see a countdown and progress indicator for my active goals so that I stay on track.

**US-072**
As a dancer, I want to receive a nudge notification when I am falling behind on a goal so that I can adjust my practice schedule.

**US-073**
As a dancer, I want to mark a goal as complete and receive a celebration so that completing goals feels meaningful.

---

## 9. Competition Tracking

**US-080**
As a dancer, I want to add upcoming competitions to my profile (name, date, location) so that I can prepare effectively.

**US-081**
As a dancer, I want to log the dances and events I am entering for a competition so that I can focus my preparation on the right material.

**US-082**
As a dancer, I want to record competition results (placement, recall, marks) after an event so that I have a record of my competitive history.

**US-083**
As a dancer, I want to view a history of all competitions I have attended so that I can see my competitive growth over time.

**US-084**
As a dancer, I want to share a competition result card to social media so that I can celebrate placements with friends and family.

---

## 10. Instructor & Notes

**US-090**
As a dancer, I want to add my instructor's name to my profile so that my progress is associated with my training context.

**US-091**
As a dancer, I want to tag coaching notes on a pattern with my instructor's name so that I know who gave me each piece of feedback.

**US-092**
As a dancer, I want to pin a note to the top of a pattern so that the most important correction is always the first thing I see.

**US-093**
As a dancer, I want to search my notes across all dances and patterns by keyword so that I can quickly find specific coaching feedback.

---

## 11. Notifications & Reminders

**US-100**
As a dancer, I want to set a weekly practice reminder (day and time) so that I am prompted to log a session.

**US-101**
As a dancer, I want to receive a notification when I am close to unlocking an achievement so that I am motivated to finish it.

**US-102**
As a dancer, I want to receive a notification when an upcoming competition is approaching so that I have time to prepare.

**US-103**
As a dancer, I want to manage all notification preferences in one place so that I am not overwhelmed with alerts I do not want.

---

## 12. Dashboard & Navigation

**US-110**
As a dancer, I want to see a home dashboard that shows my active dances, recent XP, current streak, and next milestone so that I get a complete picture in one glance.

**US-111**
As a dancer, I want the site to be fully usable on a mobile phone (thumb-friendly layout, large tap targets) so that I can use it in a studio between lessons.

**US-112**
As a dancer, I want a bottom navigation bar on mobile with quick access to Dashboard, Dances, Practice Log, Achievements, and Profile so that navigation is always one tap away.

**US-113**
As a dancer, I want the site to work offline for read access and queue writes for when I reconnect so that poor studio Wi-Fi does not interrupt my workflow.

**US-114**
As a dancer, I want to be able to install the site as a Progressive Web App (PWA) on my home screen so that it feels like a native app.

---

## Acceptance Criteria Notes

- All user-facing text must be legible at default mobile font sizes (min 16px body).
- Tap targets must be at least 44×44px per WCAG 2.1 guidelines.
- Achievement cards generated for social sharing should render at a minimum of 1080×1080px.
- PDF reports must be print-ready (A4 / US Letter) with page breaks between dance styles.
- All personal data deletion (US-006) must be irreversible and comply with GDPR / CCPA requirements.
